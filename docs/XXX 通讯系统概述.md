## 3. 各子系统详解

### 3.1 IM（即时通讯）
**核心功能**: 企业级即时消息服务

| 功能模块 | 功能点 | 说明 |
|----------|--------|------|
| **消息类型** | 文本消息 | 支持@、表情、链接预览 |
| | 富媒体消息 | 图片、语音、视频、文件 |
| | 卡片消息 | 交互式消息卡片 |
| | 系统消息 | 通知类消息 |
| **会话类型** | 单聊 | 一对一私密聊天 |
| | 群聊 | 多人群组聊天 |
| | 部门群 | 按部门自动建群 |
| | 项目群 | 临时项目协作群 |
| **消息管理** | 消息撤回 | 2 分钟内可撤回 |
| | 消息编辑 | 已发送消息可编辑 |
| | 消息置顶 | 重要消息置顶 |
| | 消息收藏 | 收藏重要消息 |
| **高级功能** | 已读未读 | 消息状态追踪 |
| | 消息搜索 | 全文搜索历史消息 |
| | 消息转发 | 转发给其他人/群 |
| | 消息抄送 | 自动抄送指定人 |
