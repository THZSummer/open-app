# 连接平台页面交互改造设计方案

**项目名称**：连接平台页面交互改造
**日期**：2026-06-04
**状态**：已批准

---

## 一、项目概述

根据交互调整文档，对 wecodesiteDemo 项目中的4个页面进行交互改造，涉及连接器管理、连接流管理及相关配置页面的功能增强和交互优化。

---

## 二、页面改造清单

### 1. connector-editor.html（连接器配置页面）

#### 1.1 页面布局调整
- **标题和描述换行展示**：将原有单行布局改为两行，标题在上一行，描述在下一行
- **新增版本列表下拉框**：位于页面头部右侧，显示版本名称、创建时间、状态标签

#### 1.2 版本控制逻辑
```javascript
// 版本状态
const VERSION_STATUS = {
  DRAFT: 'draft',      // 草稿
  PUBLISHED: 'published', // 已发布
  INVALID: 'invalid'   // 已失效
};

// 版本对应按钮
const VERSION_ACTIONS = {
  [VERSION_STATUS.DRAFT]: ['保存', '发布'],
  [VERSION_STATUS.PUBLISHED]: ['新增草稿版本', '失效'],
  [VERSION_STATUS.INVALID]: ['新增草稿版本', '删除']
};
```

#### 1.3 认证方式配置
- **新增认证类型**：Cookie、数字签名
- **Cookie认证**：参数格式与SOA一致（参数名称、参数类型、载体、描述）
- **数字签名认证**：使用固定值输入框，参数名称和载体不可编辑
- **多选支持**：允许同时选择多种认证方式

#### 1.4 入参出参配置Tab改造
- **新增三个Tab**：
  - HTTP请求头（header）
  - HTTP请求体（body）
  - URL查询参数（query）

- **参数配置项**：
  | 字段 | 类型 | 必填 | 说明 |
  |------|------|------|------|
  | 参数名称 | input | 是 | 参数标识 |
  | 参数值 | input | 否 | 参数值 |
  | 参数类型 | select | 否 | object/array/string/number/boolean |
  | 参数描述 | input | 否 | 参数说明 |

- **特殊交互**：
  - object/array类型的添加按钮改为下拉框
  - 下拉选项：添加子参数、添加同级参数
  - 删除object/array时连带删除所有子节点

#### 1.5 底部按钮固定
- 使用 `position: fixed` 固定在页面底部
- 按钮行高度：56px
- 包含按钮：返回、保存

---

### 2. flow-list.html（连接流列表页面）

#### 2.1 操作列结构调整
- **改造前**：编辑、配置、停止/启动、删除（分散展示）
- **改造后**：编辑、配置、[更多 ▼]（下拉菜单）

#### 2.2 更多下拉菜单设计
| 菜单项 | 功能说明 |
|--------|---------|
| 复制流 | 调用复制接口，创建新流数据，刷新列表 |
| 复制ID | 复制流ID到剪贴板，显示Toast提示"ID已复制" |
| 部署 | 弹出版本选择弹窗 |
| 停止/启动 | 根据当前状态切换 |
| 删除 | 删除当前连接流 |

#### 2.3 部署弹窗
- 展示当前连接流所有版本列表
- Radio单选选择部署版本
- 底部按钮：取消、确认部署

---

### 3. flow-editor.html（连接流编排页面）

#### 3.1 编辑器头部改造
```
┌──────────────────────────────────────────────────────────────┐
│ [← 返回] 连接流名称  │ 版本: [v1.0.0 草稿 ▼] │ [按钮组...]  │
└──────────────────────────────────────────────────────────────┘
```

#### 3.2 版本状态与按钮配置

| 版本状态 | 展示按钮 | 画布状态 | 节点状态 |
|---------|---------|---------|---------|
| 草稿 | 更多配置、调试、保存、发布 | 可编辑 | 可修改 |
| 已发布 | 新增草稿、更多配置、调试、失效 | 不可编辑 | 只读 |
| 已失效 | 新增草稿、更多配置、删除 | 不可编辑 | 只读 |
| 审批中 | 更多配置、撤回 | 不可编辑 | 只读 |
| 已驳回 | 新增草稿、更多配置、删除 | 不可编辑 | 只读 |

#### 3.3 更多配置抽屉
**内容**：
- **限流配置**：下拉选项 10/20/50/100（请求/秒）
- **缓存配置**：
  - 开关：开启/关闭
  - 开启时显示：
    - 缓存时间（秒）：数字输入框
    - 缓存Key：从触发器入参中选择并拼接

#### 3.4 调试弹窗
- 获取触发器节点的入参配置
- 参数名称和类型不可编辑
- 仅允许填写参数值
- 底部"立即调试"按钮
- 执行输出区域展示调试结果

#### 3.5 新增数据处理节点
```javascript
{
  type: 'data',
  name: '数据处理',
  desc: '数据转换和处理节点',
  icon: '🔮',
  color: '#722ed1'  // 紫色渐变
}
```

#### 3.6 触发器节点新增SYSTOKEN配置
- 新增配置项：SYSTOKEN凭证白名单
- 支持添加多个SYSTOKEN凭证
- 每个凭证包含：凭证名称、凭证值
