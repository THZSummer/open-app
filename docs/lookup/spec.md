## 2. 功能概述

### 2.1 背景

LookUp管理是系统的基础数据管理模块，用于统一管理业务系统中各类枚举值和配置项。通过标准化的管理机制，实现业务常量的集中维护和动态配置。

### 2.2 目标

#### 1. LookUp分类管理

**查看**：分类列表（分类编码、分类名称、路径、描述、状态、创建人/时间、修改人/时间）；支持按编码、名称、描述、状态多条件筛选

**新增**：填写分类编码、分类名称、路径（可选）、描述（可选）

**修改**：可修改分类名称、路径、描述；可切换状态（生效↔失效），需二次确认

**删除**：删除分类需二次确认，级联删除其下所有LookUp项

**导入导出**：支持异步批量导入/导出分类及其下所有LookUp项为Excel，可在任务中心查看任务进度和结果（成功数、失败数、失败明细）

#### 2. LookUp项管理

**查看**：在分类下查看LookUp项列表（项编码、项名称、项值、描述、排序、状态、创建人/时间、修改人/时间）；按编码、名称、状态筛选；详情面板查看完整信息（含扩展属性1-6）

**新增**：填写项编码、项名称、项值（可选）、排序（可选，默认0）、描述（可选）、扩展属性1-6（可选）

**修改**：可修改项名称、项值、排序、描述、扩展属性1-6；可切换状态（生效↔失效），需二次确认

**删除**：删除LookUp项需二次确认

## 3. 用户故事

### US-001: LookUp分类管理

作为系统管理员，我可以通过分类管理功能将LookUp项按业务模块进行组织，并批量导入/导出LookUp项数据。

- **查看**：查看所有分类列表；按编码、名称、描述、状态多条件筛选
- **新增**：填写分类编码、名称、路径（可选）、描述（可选）创建新分类
- **修改**：修改分类名称、路径、描述；切换状态（生效↔失效），需二次确认
- **删除**：删除分类需二次确认，级联删除其下所有LookUp项
- **导入导出**：异步批量导入/导出分类及其下所有LookUp项，可在任务中心查看进度和结果

### US-002: LookUp项管理

作为系统管理员，我可以在LookUp分类下管理LookUp项，维护业务系统中使用的枚举值和配置项。

- **查看**：在分类下查看LookUp项列表；按编码、名称、状态筛选；在详情面板查看完整信息
- **新增**：填写项编码、名称、项值（可选）、排序（可选，默认0）、描述（可选）、扩展属性1-6（可选）
- **修改**：修改项名称、项值、排序、描述、扩展属性1-6；切换状态（生效↔失效），需二次确认
- **删除**：删除LookUp项需二次确认

## 4. 功能需求详情

### 4.1 LookUp分类管理

#### FR-001: 分类列表

| 项目 | 内容 |
|------|------|
| **展示列** | 分类编码、分类名称、路径、描述、状态、创建人、创建时间、修改人、修改时间 |
| **操作列** | 编辑、生效/失效、删除 |
| **约束规则** | 按创建时间倒序；删除需二次确认并级联删除下所有项 |

#### FR-002: 新增分类

| 项目 | 内容 |
|------|------|
| **功能描述** | 创建新的LookUp分类 |
| **操作能力** | 填写分类编码、分类名称、路径、描述并提交保存 |
| **约束规则** | 分类编码必填，1-100字符，支持字母、数字、下划线、点、横杠，同一路径下唯一；分类名称必填，1-100字符；路径可选，0-100字符；描述可选，0-4000字符 |

#### FR-003: 编辑分类

| 项目 | 内容 |
|------|------|
| **功能描述** | 修改已有分类的信息 |
| **操作能力** | 修改分类名称、路径、描述；分类编码不可修改 |
| **约束规则** | 分类名称必填，1-100字符；路径可选，0-100字符；描述可选，0-4000字符 |

#### FR-004: 删除分类

| 项目 | 内容 |
|------|------|
| **功能描述** | 删除指定分类及其下所有LookUp项 |
| **操作能力** | 二次确认后执行删除 |
| **约束规则** | 物理删除不可恢复；级联删除其下所有LookUp项 |

#### FR-005: 分类状态变更

| 项目 | 内容 |
|------|------|
| **功能描述** | 切换分类的有效/失效状态 |
| **操作能力** | 一键切换分类状态（有效↔失效） |
| **约束规则** | 状态变更需二次确认，立即生效；分类失效不影响其下LookUp项状态 |

#### FR-012: 批量导入LookUp项

| 项目 | 内容 |
|------|------|
| **功能描述** | 支持通过Excel文件批量导入分类及其下所有LookUp项，异步处理 |
| **操作能力** | 下载导入模板；上传Excel文件；执行导入；提交后返回任务ID，可在任务中心查看任务进度和结果 |
| **约束规则** | 单次导入最大1000条 |
| | Excel包含分类数据和LookUp项数据两部分： |
| | - 分类数据：分类编码（必填，支持字母、数字、下划线、点、横杠，同一路径下唯一）、路径、描述、状态 |
| | - LookUp项数据：项编码（必填，同分类下唯一）、项名称、项值、排序、描述、状态、扩展属性1~6 |
| | 如分类已存在则跳过新增分类；LookUp项如已存在则自动跳过；失败记录不影响成功记录；异步处理，提交后返回任务ID |

#### FR-013: 导出LookUp项

| 项目 | 内容 |
|------|------|
| **功能描述** | 支持将分类及其下所有LookUp项导出为Excel文件，异步处理 |
| **操作能力** | 选择导出范围（指定分类或全部）；按状态、项编码、项名称筛选导出；执行导出，提交后返回任务ID，可在任务中心查看任务进度和下载结果；优先级：勾选>筛选条件>全部导出 |
| **约束规则** | 单次导出最大1000条 |
| | Excel包含分类数据和LookUp项数据两部分： |
| | - 分类数据：分类编码、路径、描述、状态 |
| | - LookUp项数据：项编码、项名称、项值、描述、排序、状态、扩展属性1~6 |
| | 文件名格式为LookUp_{分类编码}_{时间戳}.xlsx；按排序升序排列；异步处理，用户在任务页面下载 |

### 4.2 LookUp项管理

#### FR-006: LookUp项列表

| 项目 | 内容 |
|------|------|
| **展示列** | 项编码、项名称、项值、描述、排序、状态、创建人、创建时间、修改人、修改时间 |
| **操作列** | 编辑、生效/失效、删除 |
| **约束规则** | 按排序升序排列 |

#### FR-007: 新增LookUp项

| 项目 | 内容 |
|------|------|
| **功能描述** | 在指定分类下创建新的LookUp项 |
| **操作能力** | 填写项编码、项名称、项值、排序、描述、扩展属性1-6并提交保存 |
| **约束规则** | 项编码必填，1-100字符，支持字母、数字、下划线、点、横杠，同一分类下唯一；项名称必填，1-100字符；项值可选，0-2000字符；排序可选整数；描述可选，0-4000字符；扩展属性1-6可选，0-4000字符；状态默认有效 |

#### FR-008: 编辑LookUp项

| 项目 | 内容 |
|------|------|
| **功能描述** | 修改已有LookUp项的信息 |
| **操作能力** | 修改项名称、项值、排序、描述、扩展属性1-6；切换状态（生效↔失效），需二次确认；项编码和所属分类不可修改 |
| **约束规则** | 项名称必填，1-100字符；项值可选，0-2000字符；排序可选整数；描述可选，0-4000字符；扩展属性1-6可选，0-4000字符 |

#### FR-009: 删除LookUp项

| 项目 | 内容 |
|------|------|
| **功能描述** | 删除指定的LookUp项 |
| **操作能力** | 二次确认后执行删除 |
| **约束规则** | 物理删除不可恢复 |

#### FR-010: LookUp项状态变更

| 项目 | 内容 |
|------|------|
| **功能描述** | 切换LookUp项的有效/失效状态 |
| **操作能力** | 一键切换项状态（有效↔失效） |
| **约束规则** | 状态变更需二次确认，立即生效 |

#### FR-011: LookUp项详情查看

| 项目 | 内容 |
|------|------|
| **功能描述** | 查看LookUp项的完整信息 |
| **展示内容** | 基本信息（所属分类、项编码、项名称、项值、排序、状态、描述）、扩展属性1-6、操作记录（创建人、创建时间、修改人、修改时间） |
| **操作能力** | 在详情面板查看；点击编辑切换到编辑模式；关闭面板 |
