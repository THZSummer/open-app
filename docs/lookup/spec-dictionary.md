## 2. 功能概述

### 2.1 背景

数据字典管理是系统基础配置模块，用于统一管理系统中的各类属性配置和国际化文本。通过path路径实现层级归类，通过language字段支持中英文多语言，采用单表扁平结构，便于快速检索和灵活扩展。

### 2.2 目标

**查看**：数据字典列表（编码、名称、路径、描述、语言、状态、创建人/时间、修改人/时间）；支持按编码、名称、路径、语言、状态多条件筛选

**新增**：填写编码、名称、路径（可选）、描述（可选）、语言

**修改**：可修改名称、路径、描述；可切换状态（生效↔失效），需二次确认；编码不可修改

**删除**：删除数据字典需二次确认

**导入导出**：支持异步批量导入/导出数据字典为Excel，可在任务中心查看任务进度和结果（成功数、失败数、失败明细）

## 4. 功能需求详情

### 4.1 数据字典管理

#### FR-001: 数据字典列表

| 项目 | 内容 |
|------|------|
| **展示列** | 编码、名称、路径、描述、语言、状态、创建人、创建时间、修改人、修改时间 |
| **操作列** | 编辑、生效/失效、删除 |
| **筛选条件** | 编码、名称、路径、语言、状态 |
| **约束规则** | 按创建时间倒序 |

#### FR-002: 新增数据字典

| 项目 | 内容 |
|------|------|
| **功能描述** | 创建新的数据字典条目 |
| **操作能力** | 填写编码、名称、路径、描述、语言并提交保存 |
| **约束规则** | 编码必填，1-100字符，支持字母、数字、下划线、点、横杠，同一路径下唯一；名称必填，1-100字符；路径可选，0-100字符；描述可选，0-4000字符；语言必填，默认中文；状态默认有效 |

#### FR-003: 编辑数据字典

| 项目 | 内容 |
|------|------|
| **功能描述** | 修改已有数据字典的信息 |
| **操作能力** | 修改名称、路径、描述；编码不可修改 |
| **约束规则** | 名称必填，1-100字符；路径可选，0-100字符；描述可选，0-4000字符 |

#### FR-004: 删除数据字典

| 项目 | 内容 |
|------|------|
| **功能描述** | 删除指定数据字典条目 |
| **操作能力** | 二次确认后执行删除 |
| **约束规则** | 物理删除不可恢复 |

#### FR-005: 数据字典状态变更

| 项目 | 内容 |
|------|------|
| **功能描述** | 切换数据字典的有效/失效状态 |
| **操作能力** | 一键切换状态（有效↔失效） |
| **约束规则** | 状态变更需二次确认，立即生效 |

#### FR-006: 批量导入数据字典

| 项目 | 内容 |
|------|------|
| **功能描述** | 支持通过Excel文件批量导入数据字典，异步处理 |
| **操作能力** | 下载导入模板；上传Excel文件；执行导入；提交后返回任务ID，可在任务中心查看任务进度和结果 |
| **约束规则** | 单次导入最大1000条 |
| | Excel列：编码（必填，支持字母、数字、下划线、点、横杠，同路径下唯一）、名称、路径、描述、语言、状态 |
| | 如已存在（同路径同编码）则自动跳过；失败记录不影响成功记录；异步处理，提交后返回任务ID |

#### FR-007: 导出数据字典

| 项目 | 内容 |
|------|------|
| **功能描述** | 支持将数据字典导出为Excel文件，异步处理 |
| **操作能力** | 选择导出范围；按编码、名称、路径、语言、状态筛选导出；执行导出，提交后返回任务ID，可在任务中心查看任务进度和下载结果；优先级：勾选>筛选条件>全部导出 |
| **约束规则** | 单次导出最大1000条 |
| | Excel列：编码、名称、路径、描述、语言、状态 |
| | 文件名格式为Dictionary_{路径}_{时间戳}.xlsx；异步处理，用户在任务页面下载 |
