## 核心内容摘要

### 能力分类模型
- **平台本身能力**（内部管理）：应用管理、成员管理、API 管理、事件管理、权限管理等
- **开放给三方的能力**：API 调用、事件订阅、回调通知、数据查询等

### 目标用户
- **能力 Owner**: 注册能力、维护能力文档

- **三方平台业务方**: 发现能力、订阅能力、消费能力

### 核心流程
