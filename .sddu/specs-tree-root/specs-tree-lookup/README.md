## 核心功能

### LookUp 分类管理
| 功能 | 说明 |
|------|------|
| 分类列表 | 按编码、名称、描述、状态多条件筛选 |
| 新增分类 | 填写分类编码、分类名称、路径（可选）、描述（可选） |
| 修改分类 | 可修改名称、路径、描述；切换状态需二次确认 |
| 删除分类 | 删除需二次确认，级联删除其下所有 LookUp 项 |
| 导入导出 | 异步批量导入/导出分类及项为 Excel |

### LookUp 项管理
| 功能 | 说明 |
|------|------|
| 项列表 | 在分类下查看；按编码、名称、状态筛选；详情面板查看完整信息（含扩展属性 1-6） |
| 新增项 | 填写项编码、项名称、项值（可选）、排序、描述、扩展属性 1-6（可选） |
| 修改项 | 可修改项名称、项值、排序、描述、扩展属性 1-6；切换状态需二次确认 |
| 删除项 | 删除需二次确认 |
