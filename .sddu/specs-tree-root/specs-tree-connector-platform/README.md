# 连接器平台

**Feature ID**: CONN-PLAT-001  
**状态**: specified  

## 概述

连接器平台是开放平台的组成部分，提供与 API、事件、回调同级并列的第四种开放形式。通过连接器封装和连接流可视化编排，将三方平台消费开放能力时的人工编码场景转化为低代码/零代码配置，使原有开放更便捷。
