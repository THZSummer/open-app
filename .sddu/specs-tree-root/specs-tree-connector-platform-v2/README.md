## 三大升级方向详情

### G1: 多版本及生命周期
- 5 状态状态机：草稿→已发布→已弃用→已退役→已归档
- SemVer 规则 + 递增整数版本号
- 版本快照/对比/回滚能力
- 双凭据轮转零停机机制
- 全量审计日志

### G2: 连接配置增强
- 7 种认证方式（含 mTLS）
- 三级超时模型（连接/读取/请求）
- 令牌桶限流（QPS/并发数）
- Fork-Join 并行编排（≤ 5 分支）
- 3 种失败/合并策略

### G3: 运行时与监控
- 执行历史（7 状态，30 天保留）
- 5 级可配运行日志
- 重试 + 指数退避 + 熔断器 + 死信队列
- 运行记录与执行步骤存储
