### 2.1 AppList - 应用列表页面

- **文件路径**: `src/pages/AppList/AppList.jsx`
- **目的**: 显示用户的应用列表，提供创建新应用的入口
- **路由**: `/` (根路径，应用首页)

#### 布局结构
```
┌─────────────────────────────────────────────────┐
│                  页面容器 (max-width: 960px)     │
├─────────────────────────────────────────────────┤
│  ┌─────────────────────┐  ┌──────────────────┐ │
│  │     我的应用       │  │   创建应用 按钮   │ │
│  │   (h2, 20px)     │  │   (Primary)       │ │
│  └─────────────────────┘  └──────────────────┘ │
├─────────────────────────────────────────────────┤
│         应用卡片网格 (3列, gap: 12px)            │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
│  │ AppCard │  │ AppCard │  │ AppCard │        │
│  │ 3列    │  │        │  │        │        │
│  └─────────┘  └─────────┘  └─────────┘        │
└─────────────────────────────────────────────────┘
```

#### 内部布局详情
- **外层容器**: `max-width: 960px`, `margin: 0 auto`, `min-height: 100%`
- **头部区域**: `display: flex`, `justify-content: space-between`, `margin-bottom: 24px`
- **标题样式**: `font-size: 20px`, `font-weight: 600`, `color: var(--text-primary)`
- **内容网格**: `display: grid`, `grid-template-columns: repeat(3, 1fr)`, `gap: 12px`

#### 样式规格
- **Colors**: `var(--text-primary)`, `var(--primary-color)`, `var(--primary-hover)`
- **Typography**: 标题 `20px/600`, 按钮 `18px`
- **Spacing**: `margin-bottom: 24px`, `gap: 12px`
- **Dimensions**: `max-width: 960px`

#### 交互行为
- **点击 "创建应用"**: 打开创建应用弹窗 (CreateAppModal)

#### 路由行为
- **路径**: `/`
- **URL参数**: `appId` (用于跳转到应用详情)

## 5. 文档规则 (IMPORTANT)

### 5.1 Layout Components
- **DO NOT** use Ant Design's `Row`, `Col`, `Space`, or `Typography` components for layout and typography
- Use standard HTML elements (div, span, etc.) with CSS/Less for layout instead
- Use native CSS flexbox or grid for positioning elements

### 5.2 Pagination
- **DO NOT** use Ant Design Table's built-in pagination
- Always use separate `Pagination` component from Ant Design
- Pagination must include:
  - `showQuickJumper`: Enable quick page number input
  - `showSizeChanger`: Enable page size selector
  - `pageSizeOptions`: `[10, 20, 50]`
  - Default `pageSize`: 10
- Place the Pagination component below the Table, outside of the Table component

Example:
```jsx
<Table columns={columns} dataSource={data} rowKey="id" pagination={false} />
<div style={{ marginTop: 16, textAlign: 'right' }}>
  <Pagination
    total={total}
    current={currentPage}
    pageSize={pageSize}
    pageSizeOptions={[10, 20, 50]}
    showSizeChanger
    showQuickJumper
    showTotal={(total) => `共 ${total} 条`}
    onChange={handlePageChange}
  />
</div>
```
