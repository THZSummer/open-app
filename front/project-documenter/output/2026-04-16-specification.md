# 飞书开放平台开发者控制台 - 项目规格文档

## 1. 项目概述

**项目名称**: 飞书开放平台开发者控制台 (Feishu Developer Console)  
**项目版本**: 1.0.0  
**项目描述**: 一个基于 React + Ant Design 的飞书开放平台管理后台，用于管理第三方应用的生命周期，包括应用创建、基础信息配置、权限管理、事件订阅、成员管理等核心功能。

## 2. 技术栈

### 前端框架
- **React**: ^18.2.0
- **React Router**: ^6.20.0 (路由管理)
- **Ant Design**: ^4.24.16 (UI 组件库)
- **@ant-design/icons**: ^6.1.1 (图标库)

### 构建工具
- **Vite**: ^5.0.0
- **@vitejs/plugin-react**: ^4.2.0
- **Less**: ^4.2.0 (CSS 预处理器)

### 样式方案
- Less 模块化样式 (`.m.less`)
- CSS 变量管理全局主题

## 5. 路由配置

### 路由结构 (App.jsx)

```
/                          → AppList (我的应用 - 首页)
/basic-info                → BasicInfo (凭证和基础信息)
/members                   → Members (成员管理)
/capabilities              → Capabilities (添加应用能力)
/capability-detail        → CapabilityDetail (能力详情)
/api-management            → ApiManagement (API管理)
/events                    → Events (事件配置)
/operation-log             → OperationLog (操作日志)
/version-release          → VersionRelease (版本发布)
/version-release/form     → VersionForm (版本表单)
```

### URL 参数规范
- `appId`: 应用 ID (例: `/basic-info?appId=1`)
- `caps`: 启用的应用能力列表，逗号分隔 (例: `?caps=bot,web`)
- `type`: 应用能力类型 (例: `?type=bot`)

## 12. 交互规范

### 分页组件规范
所有列表页面使用独立的 Pagination 组件（非 Ant Design Table 内置分页）：
```javascript
<Pagination
  total={total}
  current={currentPage}
  pageSize={pageSize}
  pageSizeOptions={[10, 20, 50]}
  showSizeChanger
  showQuickJumper
  showTotal={(total) => `共 ${total} 条`}
  onChange={handlePageChange}
/>
```

### 弹窗规范
- Modal 组件使用 `destroyOnClose` 确保关闭时销毁
- 抽屉组件放在页面级别，方便状态管理

### 路由参数传递
- 使用 `useSearchParams` 获取 URL 参数
- 导航时携带 `appId` 和 `caps` 参数
- 保持参数一致性
